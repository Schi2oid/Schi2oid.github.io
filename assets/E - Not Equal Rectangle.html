    E - Not Equal Rectangle Editorial /

    Time Limit: 2 sec / Memory Limit: 1024 MB

    配点 : 800800800 点

    問題文

    N×MN \times MN×M のマス目があり、あなたはこれから全てのマスに 111 以上 252525 以下の整数を 111 つずつ書き込みます。上から iii 行目、左から jjj 列目のマスに書き込む整数を ai,ja_{i,j}ai,j​ とします。

    以下の条件を満たす整数の書き込み方を一つ求めてください。本問題の制約下で、条件を満たす整数の書き込み方が必ず存在することが証明できます。

    • 任意の整数 1≤x1<x2≤N,1≤y1<y2≤M1\leq x_1 < x_2\leq N,1\leq y_1 < y_2 \leq M1≤x1​<x2​≤N,1≤y1​<y2​≤M について、ax1,y1,ax1,y2,ax2,y1,ax2,y2a_{x_1,y_1},a_{x_1,y_2},a_{x_2,y_1},a_{x_2,y_2}ax1​,y1​​,ax1​,y2​​,ax2​,y1​​,ax2​,y2​​ が全て一致してはならない。

    制約

    • 2≤N,M≤5002 \leq N , M \leq 5002≤N,M≤500
    • 入力は全て整数

    入力

    入力は以下の形式で標準入力から与えられる。

    NNN MMM

    出力

    条件を満たす書き込み方の 111 つを、以下の形式で出力せよ。

    a1,1a_{1,1}a1,1​ a1,2a_{1,2}a1,2​ …\ldots… a1,Ma_{1,M}a1,M​
a2,1a_{2,1}a2,1​ a2,2a_{2,2}a2,2​ …\ldots… a2,Ma_{2,M}a2,M​
⋮\vdots⋮
aN,1a_{N,1}aN,1​ aN,2a_{N,2}aN,2​ …\ldots… aN,Ma_{N,M}aN,M​

    条件を満たす解が複数存在する場合、どれを出力しても正解とみなされる。

    入力例 1 Copy

    Copy
    2 3

    出力例 1 Copy

    Copy
    1 1 1
1 2 3

    (x1,x2,y1,y2)(x_1,x_2,y_1,y_2)(x1​,x2​,y1​,y2​) の組として考えられるのは (1,2,1,2),(1,2,2,3),(1,2,1,3)(1,2,1,2),(1,2,2,3),(1,2,1,3)(1,2,1,2),(1,2,2,3),(1,2,1,3) の 333 つです。

    どの組についても 444 マスに書かれた数字が全て一致してはいないので、この出力は条件を満たします。

    Score : 800800800 points

    Problem Statement

    We have a grid with N×MN \times MN×M squares. You will fill every square with an integer between 111 and 252525 (inclusive). Let ai,ja_{i,j}ai,j​ be the integer to be written in the square at the iii-th row from the top and jjj-th column from the left.

    Find a way to fill the squares to satisfy the condition below. It can be proved that, under the Constraints of this problem, such a way always exists.

    • For any integers 1≤x1<x2≤N,1≤y1<y2≤M1\leq x_1 < x_2\leq N,1\leq y_1 < y_2 \leq M1≤x1​<x2​≤N,1≤y1​<y2​≤M, it must not be the case that ax1,y1,ax1,y2,ax2,y1,ax2,y2a_{x_1,y_1},a_{x_1,y_2},a_{x_2,y_1},a_{x_2,y_2}ax1​,y1​​,ax1​,y2​​,ax2​,y1​​,ax2​,y2​​ are all equal.

    Constraints

    • 2≤N,M≤5002 \leq N , M \leq 5002≤N,M≤500
    • All values in input are integers.

    Input

    Input is given from Standard Input in the following format:

    NNN MMM

    Output

    Print one way to fill the squares to satisfy the condition, in the format below:

    a1,1a_{1,1}a1,1​ a1,2a_{1,2}a1,2​ …\ldots… a1,Ma_{1,M}a1,M​
a2,1a_{2,1}a2,1​ a2,2a_{2,2}a2,2​ …\ldots… a2,Ma_{2,M}a2,M​
⋮\vdots⋮
aN,1a_{N,1}aN,1​ aN,2a_{N,2}aN,2​ …\ldots… aN,Ma_{N,M}aN,M​

    If there are multiple solutions, printing any of them will be accepted.

    Sample Input 1 Copy

    Copy
    2 3

    Sample Output 1 Copy

    Copy
    1 1 1
1 2 3

    (x1,x2,y1,y2)(x_1,x_2,y_1,y_2)(x1​,x2​,y1​,y2​) can be one of (1,2,1,2),(1,2,2,3),(1,2,1,3)(1,2,1,2),(1,2,2,3),(1,2,1,3)(1,2,1,2),(1,2,2,3),(1,2,1,3).

    For any of them, the numbers written in the squares are not all equal, so this output satisfies the condition.
